Urubu

A micro CMS for static websites, with a focus on good navigation practices.

27-Feb-2014: Urubu 0.3 released

For websites, not just blogs

Set up a website as logically related content. Content ordering is fully flexible, not just by date.

Markdown for content

Enter content in light-weight Markdown syntax. Popular extensions are enabled, including support for source code.

Focus on navigation

Easily set up a navbar, a table of contents sidebar, breadcrumbs and pagers. Refer to other pages with simple syntax, like in wikis.

Built with Python

Urubu is open source and built with Python. It uses the powerful Jinja2 library for templating.

Ready for Bootstrap

Urubu plays well with Bootstrap, the popular framework to develop themes for great looking websites.

Ideal with git or mercurial

Develop your website in a git or mercurial repo and use the same workflow as for coding projects. Deploy by pushing.
